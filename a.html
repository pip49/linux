1과목: 리눅스 일반 (기초 뼈대)

개요: 리눅스의 역사, 철학 및 가장 기초적인 파일/디렉터리 조작 명령어를 다룹니다. 1차 온라인 시험의 핵심이자 2차 시험의 기반이 됩니다.

📌 필수 암기 디렉터리 구조 (FHS)

/etc시스템의 각종 설정 파일이 저장되는 곳 (가장 중요)
/var로그(Log), 메일 등 내용이 자주 변하는 가변 데이터 저장
/bin사용자가 사용하는 기본적인 필수 명령어 위치
/sbin시스템 관리자(root)가 사용하는 시스템 관리 명령어

📌 시험에 무조건 나오는 기초 명령어

  • ls: 디렉터리 목록 보기 (-a: 숨김파일 표시, -l: 상세 정보 표시)
  • cp / mv / rm: 파일 복사 / 이동(이름변경) / 삭제 (-r: 디렉터리 대상 적용)
  • pwd: 현재 작업 중인 디렉터리의 절대 경로 표시
  • chmod: 파일 권한 변경 (예: chmod 755 파일명 -> rwxr-xr-x)
  • chown: 파일의 소유자와 소유 그룹 변경

2과목: 리눅스 운영 및 관리 (실무 제어)

개요: 사용자(User) 관리, 프로세스 관리, 파일 시스템 구축, 소프트웨어 설치 등 서버 관리자로서 시스템을 운영하는 실제적인 기술을 다룹니다. 2차 시험에서 가장 높은 배점(48문항)을 차지합니다.

📌 필수 암기 핵심 파일 경로

/etc/passwd사용자 계정 정보 (UID, GID, 기본 셸 등)
/etc/shadow사용자의 암호화된 비밀번호 및 만료일 관리
/etc/fstab부팅 시 자동 마운트할 파일 시스템 정보 설정

📌 단골 출제 핵심 명령어 및 옵션

  • ps: 현재 실행 중인 프로세스 확인 (ps aux 또는 ps -ef 형태로 자주 출제)
  • kill: 프로세스 종료 (가장 강한 강제 종료 시그널은 -9)
  • tar: 파일 아카이브 묶기/풀기 (-xvf: 풀기, -cvf: 묶기, -z: gzip 압축 연동)
  • rpm / yum / apt: 패키지 설치 및 관리 명령어 (각 진영별 차이점 비교 출제)
  • cron: 정기적인 예약 작업 설정 (crontab -e로 편집, 분-시-일-월-요일 순서)

3과목: 리눅스 활용 (네트워크 & 서비스)

개요: 리눅스 서버에 네트워크를 연결하고, 웹 서버(Apache), 파일 서버(FTP, Samba) 등 네트워크 기반의 오픈소스 서비스를 활성화하고 제어하는 과목입니다. (32문항 출제)

📌 서비스별 프로토콜 및 기본 포트(Port) 번호

SSH (보안 원격 접속)TCP 22번 (설정 파일: /etc/ssh/sshd_config)
HTTP (웹 서버 / Apache)TCP 80번 (설정 파일: httpd.conf)
FTP (파일 전송)TCP 21번(제어), 20번(데이터)
DNS (도메인 네임 서버)UDP/TCP 53번 (설정 파일: named.conf)

📌 네트워크 확인 및 설정 명령어

  • ifconfig / ip: 네트워크 인터페이스(IP 주소 등) 확인 및 설정
  • netstat / ss: 현재 시스템의 네트워크 연결 상태 및 열려 있는 포트 확인
  • ping: 네트워크 연결이 정상적으로 작동하는지 패킷을 보내 확인
  • nslookup / dig: DNS 서버에 도메인 이름 쿼리(IP 주소 조회)
