리눅스 마스터 2급 2차 시험 개념 트리

PART 01. 리눅스 일반 (24문항)
1. 리눅스의 이해
  • 운영체제의 이해: 커널(Kernel)과 셸(Shell)의 역할 구조, 유닉스(UNIX)와의 연관성 및 미닉스(Minix) 기반 개발 역사
  • 리눅스의 철학 및 역사: GNU 프로젝트(리차드 스톨만)와 FSF, 리누스 토발즈의 커널 공개
  • 오픈소스 라이선스: GPL(수정본 공개 필수), LGPL(라이브러리 링크 허용), BSD/Apache(상용화 허용, 출처 명시)
  • 리눅스 배포판 계열: RedHat 계열(CentOS, Fedora, Rocky), Debian 계열(Ubuntu, Debian), Slackware 계열(OpenSUSE)
2. 리눅스 설치
  • 하드웨어 선택 및 파티션: 파티션 개념(주 파티션 최대 4개, 논리 파티션), Swap 파티션(메모리 부족 백업 공간, RAM의 1~2배 설정)
  • 부트 매니저: GRUB의 특징 및 설정 파일 구조, 부팅 프로세스 순서(BIOS -> MBR -> GRUB -> Kernel -> systemd)
3. 기본 명령어 및 셸 제어
  • 디렉터리 구조 (FHS 표준): /etc(설정), /var(가변 로그), /bin(필수 명령어), /sbin(관리자 명령어), /proc(가상 프로세스 정보)
  • 기본 조작 명령어: ls(-a, -l, -R 옵션), cp(-r 옵션), mv, rm(-f, -r 옵션), pwd, mkdir, rmdir
  • 텍스트 처리 및 편집 명령어: cat, more, less, head, tail(-f 실시간 로그 확인), grep(-i, -v, -n 옵션), find(-name, -user, -perm 옵션)
  • 리다이렉션 및 파이프: 표준 입력(0), 표준 출력(1), 표준 에러(2), 덮어쓰기(>), 이어쓰기(>>), 에러 추출(2>), 명령어 연동 파이프(|)
  • Bash 셸 환경 변수: env, export, 대소문자 매칭 필수 환경변수($PATH, $HOME, $USER, $SHELL, $HISTSIZE)
PART 02. 리눅스 운영 및 관리 (48문항 - 최고 배점)
1. 사용자 및 그룹 관리
  • 계정 정보 파일: /etc/passwd(7개 필드 구조: UID, GID, 홈디렉터리, 셸 경로), /etc/shadow(비밀번호 암호화 및 만료 주기)
  • 그룹 정보 파일: /etc/group, /etc/gshadow
  • 사용자 생성 및 설정: useradd / adduser, 생성 정책 참조 파일(/etc/default/useradd, /etc/login.defs), 환경 설정 뼈대 디렉터리(/etc/skel)
  • 계정 수정 및 삭제: usermod(-G, -s, -L, -U 옵션), userdel(-r 옵션 필수), passwd, chage(비밀번호 만료 정책 설정)
2. 파일 시스템 및 디스크 관리
  • 리눅스 파일 시스템: ext2(초기 표준), ext3(저널링 도입), ext4(대용량 및 서브디렉터리 확장), xfs(고성능 64비트, RHEL 7+ 기본값)
  • 디스크 파티션 및 포맷: fdisk(n, d, p, w 서브 명령어), mkfs / mke2fs(-t 옵션으로 파일시스템 지정)
  • 마운트 제어: mount(장치와 디렉터리 연결), umount, 자동 마운트 설정 파일(/etc/fstab의 6개 필드 구조 및 옵션)
  • 용량 점검 명령어: df(디스크 전체 남은 공간, -h 인류 친화 단위), du(특정 디렉터리/파일 사용 용량)
3. 프로세스 및 작업 제어
  • 프로세스 확인: ps 진영별 옵션(ps aux - BSD 스타일 / ps -ef - System V 스타일), top(실시간 모니터링), pstree
  • 시그널 제어: kill, killall, 필수 암기 시그널 번호(1: SIGHUP 재시작, 9: SIGKILL 강제종료, 15: SIGTERM 정상종료 기본값)
  • 작업 전환 (Job Control): 포그라운드(Foreground), 백그라운드 전환 및 기호(&), 작업 일시정지(Ctrl + Z), 작업 명세(jobs), 전환 명령어(fg, bg)
4. 소프트웨어 및 압축 관리
  • 압축 및 아카이브: tar 4대 필수 조합(-cvf 묶기, -xvf 풀기, -z gzip 연동, -j bzip2 연동, -J xz 연동)
  • RedHat 계열 패키지: rpm(-ivh 설치, -Uvh 업그레이드, -e 삭제, -qa 전체 조회, -ql 파일 경로 조회), 고도화 도구 yum / dnf(의존성 자동 해결)
  • Debian 계열 패키지: dpkg(-i 설치, -r 삭제, -l 조회), 고도화 도구 apt-get / apt
5. 장치 관리 및 시스템 예약
  • 주변 장치 설정: 쿠키 및 모듈 관리 명령어(lsmod, insmod, rmmod, modprobe)
  • 정기 스케줄러 (Cron): crontab -e(편집), crontab -l(조회), crontab -r(삭제), 설정 필드 순서(분-시-일-월-요일)
  • 단발성 스케줄러 (At): at(지정 시간 단 1회 실행), atq(대기열 조회), atrm(작업 삭제), 실행 데몬 atd
PART 03. 리눅스 활용 (32문항)
1. 네트워크 개념 및 설정
  • 네트워크 기초 프로토콜: OSI 7계층 구조와 계정별 장비(라우터, 스위치), TCP/IP 4계층 매핑 구조
  • 네트워크 인터페이스 설정: ifconfig, ip(주소 할당 및 확인), 인터페이스 활성/비활성(ifup, ifdown)
  • 인프라 환경 설정 파일: /etc/sysconfig/network-scripts/ifcfg-[인터페이스명](고정IP 설정 매개변수 BOOTPROTO=static, ONBOOT=yes), /etc/resolv.conf(nameserver DNS 지정), /etc/hosts(로컬 매핑)
  • 네트워크 상태 점검: netstat 필수 옵션 조합(-anp 모든 포트 번호와 PID 매핑, -lntp 리슨 상태 TCP 조회), ping, traceroute, route(라우팅 테이블 관리), arp(MAC 주소 확인), nslookup / dig(DNS 질의)
2. 오픈소스 서버 서비스
  • 웹 서버 (Web): HTTP 프로토콜(기본 TCP 80번), Apache 데몬(httpd), 설정 파일(httpd.conf 내부 DocumentRoot 지시어)
  • 원격 접속 서버: 보안 셸(SSH, TCP 22번, 설정 파일 /etc/ssh/sshd_config), 텔넷(Telnet, TCP 23번, 암호화 없음)
  • 파일 공유 서버: Samba(이기종 윈도우-리눅스 간 공유, 포트 139/445, smb.conf), NFS(리눅스 간 디렉터리 마운트 공유, /etc/exports)
  • 도메인 네임 서버 (DNS): BIND 데몬(named, UDP/TCP 53번), 설정 파일(/etc/named.conf 및 Zone 파일 명세)
3. 보안 및 가상화 인프라
  • 패킷 필터링 방화벽: iptables 3대 체인(INPUT, OUTPUT, FORWARD), 3대 타겟 타깃 처리 방식(ACCEPT 통과, DROP 무응답 차단, REJECT 안내 후 거절)
  • 가상화 하이퍼바이저: 전가상화(하드웨어 완전 가상화, 수정 없음, KVM, VirtualBox) 대 반가상화(게스트 OS 커널 수정, 속도 향상, Xen)
  • 컨테이너 가상화: 하이퍼바이저 없이 OS 커널 레벨 격리 공간 제공, 가볍고 빠른 배포(대표 주자: Docker)
  • 클라우드 서비스 모델: IaaS(하드웨어 자원 인프라 제공, AWS EC2), PaaS(개발 런타임 플랫폼 제공, Firebase), SaaS(완성형 소프트웨어 제공, Google Drive)
